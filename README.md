# Glitch Jeopardy (Work in Progress)

This is a playful, experimental Python game inspired by Jeopardy!  
The idea is simple: answer questions, but beware—the answers might change on you.  

## How to Play
1. Run the `glitch_jeopardy.py` script.  
2. Read the question and type your answer.  
3. Watch as the game sometimes “glitches” your answers, changing the outcome!  

## Notes
